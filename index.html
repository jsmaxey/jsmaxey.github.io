Selectors

Types of Selectors

  • HTML Elements Generally

    • Generally, you can select HTML Elements by just naming them

    ex. h1 {

    }

  • Classes of Elements

    • To select a class, put a period (.) before the class name

    ex. .subhead {

    }

  • Specifically Identified Elements

    • To select an HTML element with a specific ID, use the # before the ID name

    ex. #picture1caption {

    }
